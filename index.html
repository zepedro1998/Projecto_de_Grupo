O que é Animação na web?

É um conjunto de frames do qual na sua união criam um padrão ou uma sequencia que dá vida. Junto com a popularização da Web, animações para a mesma também foram se popularizando. GIF, uma sigla para Graphics Interchange Format (Formato de Intercâmbio de Gráficos), é um tipo de arquivo de imagem que permite que animações sejam vistas num Web Browser. Ele era o formato mais popular para animações na Web até relativamente pouco tempo. Porém, é um formato limitado, permitindo apenas 256 cores simultâneas e muitas vezes gerando arquivos grandes para animações mais complexas. Com o surgimento de outros formatos, como o Flash, criado pela Macromedia e actualmente mantido pela Adobe, ele passou a ser utilizado apenas para pequenas animações. O Flash também tem a vantagem de permitir uso de efeitos sonoros e de criar animações interativas, dentre outras. A nova versão do Flash já conta até com ferramentas para animação em 3D, que permite maiores possibilidades de efeitos nas animações.

Transcição

Como é que se cria uma Transcição no CSS?

Para criar um efeito de Transição,temos que ter em conta dois aspetos: 1.A propriedade de CSS do qual queremos adicionar o efeito em si; 2. A duração do efeito.

Comandos: transition- Transição; transition-delay- Atraso da Transição; transition-duration- Duração da Transição; transition-property- Propriedades da Transição; transition-timing-function-Tempo da Transição;

ATENÇÃO: Se a duração não for especificada, a transição não acontecerá pois o seu valor será 0.

O que são animações CSS?

Uma animação permite que um elemento mude gradualmente de um estilo para outro.

Você pode alterar quantas propriedades CSS quiser, quantas vezes quiser.

Para usar animação CSS, você deve primeiro especificar alguns keyframes para a animação.

Comandos:

  • @keyframes
  • animation-name- Nome;
  • animation-duration- Duração;
  • animation-delay- Atraso;
  • animation-iteration-count- Interação;
  • animation-direction- Direção;
  • animation-timing-function- Função;
  • animation-fill-mode- Modo de Preenchimento;
  • animation

    Os keyframes contêm os estilos que o elemento terá em determinados momentos.
